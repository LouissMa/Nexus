# 🌟 LifeAgent

> **拥有长期记忆的主动式个人 AI 操作系统。** > 它不仅仅是一个聊天机器人，而是一个能记住你的人生轨迹、跟踪你的进度，并主动协助你达成目标的数字分身。

[English](./README.md) | [中文](./README_zh.md)

---

## 💡 愿景

目前的 AI 工具都是被动式的：**你提问 → AI 回答。** LifeAgent 致力于打破这一范式。它将作为你的生活管家和数字分身，记住你的上下文，理解你的长期目标，并在你需要的时候主动发起对话。

## ✨ 核心特性

* **🧠 长期记忆引擎**：记住你的生活习惯、职业目标、人际关系和历史对话。
* **🔔 主动智能系统**：打破“你问我答”，基于你的状态主动提供提醒、建议，甚至直接执行任务。
* **🎯 个人目标追踪**：在数月甚至数年的时间跨度内，管理你的健康、学习和效率指标。

## 🚀 MVP (最小可行性产品)

第一版我们将极度克制，专注于跑通最核心的闭环：
1.  **记忆系统**：存储并检索用户的核心上下文。
2.  **目标追踪**：设定并监控用户的具体目标。
3.  **主动提醒**：AI 基于时间和上下文主动推送消息。

## 🔮 未来规划

随着用户数据的积累，LifeAgent 将演化为真正的生活管理 OS：
* 日历与任务的自动化管理
* 邮件自动分类与回复
* 形成具有极强用户粘性的“数字情感护城河”

## 🛠️ 技术栈 (规划中)

| 模块 | 技术选型 |
| :--- | :--- |
| **前端** | React / Flutter |
| **后端** | Python / Node.js |

## 🗺️ 路线图

- [ ] **Phase 1**: 搭建基础记忆系统 (Memory Engine & RAG)。
- [ ] **Phase 2**: 开发主动触发引擎 (定时任务 + LLM 逻辑)。
- [ ] **Phase 3**: 演化为完整的个人 AI Agent (接入更多外部工具)。
